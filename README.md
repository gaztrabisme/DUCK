# DUCK Landing Page

The DUCK Landing Page is a Next.js-based frontend designed to introduce users to the DUCK token currency, which rewards Telegram usage with a playful duck theme. The project focuses on creating a responsive and modular landing page structure, using placeholder elements for future graphics and illustrations.

## Overview

The project is built using Next.js, a React framework known for its fast development and optimized performance. The architecture emphasizes a responsive, modular, and semantic HTML structure with basic CSS for layout. The use of reusable React components and placeholder elements for future graphics ensures scalability and ease of content integration. The project adheres to React/Next.js best practices and is well-commented for easy maintenance.

## Features

- **Responsive Design**: The landing page is designed to be accessible and visually appealing across various devices.
- **Modular Structure**: Utilizes reusable React components for a clean and maintainable codebase.
- **Semantic HTML**: Ensures the use of semantic HTML elements for better accessibility and SEO.
- **Placeholder Elements**: Includes placeholders for future graphics and illustrations, allowing for easy content integration.
- **Navigation**: Features a fixed header with navigation links and social media icons.
- **Hero Section**: Engaging introduction with a title, tagline, and call-to-action buttons.
- **How-to-Earn Section**: Guides users through the steps to earn DUCK tokens.
- **Leaderboard and Rewards**: Displays a scrollable leaderboard and information on rewards.
- **Community Section**: Encourages community participation with user testimonials.
- **Footer**: Contains essential links and contact information.

## Getting started

### Requirements

- Node.js (version specified in `package.json`)
- A modern web browser

### Quickstart

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd <project-directory>
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser and navigate to:**
   ```
   http://localhost:3000
   ```
